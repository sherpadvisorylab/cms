Dashboard

Manage content areas, pages, components, templates, navigation, forms, emails, users, and settings.

Quick links

Use the sidebar to open each section. Data is stored in localStorage (keys as per docs).

  • Areas – List and edit areas (Basic Data, Style, Design, Legal, Tracking, Access Policy)
  • Pages – Create and manage pages (structure, content, SEO)
  • Navigation – Navigation blocks (items, display template)
  • Components – Component library (Page, UI, Navigation)
  • Templates – Reusable page structures
  • Forms – Form generator and embed {{form:id}}
  • Emails – Email templates
  • Users – User management and roles
  • Settings – System variables, branding, SSO
